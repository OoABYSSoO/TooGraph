# 最终游戏广告创意包示例

## 1. Creative Brief 摘要

游戏类型是策略 RPG，不应固定成 SLG 买量套路。创意核心是“资源紧张下的生存抉择”，用基地资源、天气灾害和队员羁绊证明策略深度。

## 2. 竞品模式和新闻上下文

可用模式包括资源三选一、暴雪夜袭和角色牺牲。市场材料提示玩家会质疑“只看战力”的假策略，因此脚本必须露出实机 UI 或明确标注概念镜头。

## 3. 推荐变体

优先制作“暴雪夜只差 1 秒”版本：30 秒内能展示天气压力、拖拽队员、基地耐久和紧张反转，且素材可得性高。

## 4. 图片/视频生成任务

- 图片：雪夜基地首帧、天气预警 UI、三名队员选择卡、结尾 CTA。
- 视频：前 3 秒暴雪断电，中段拖拽修复发电机，末尾基地防线恢复。

## 5. 审核和风险

避免血腥特写、误导性抽卡概率和夸大收益。所有概念镜头需要标注或补实机画面。

## 6. 后台记忆整理参考

可供 Buddy 后台复盘参考的稳定创意偏好：该游戏广告优先使用“高压决策 + 实机 UI 证明策略深度”的创意方向。本模板不直接写长期记忆。
