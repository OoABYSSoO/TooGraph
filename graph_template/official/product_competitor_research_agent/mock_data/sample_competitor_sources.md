# 示例竞品资料

## 竞品 A：Prompt Library

- 定位：沉淀提示词模板，支持搜索和复制。
- 功能：分类、收藏、版本说明、团队共享。
- 局限：运行过程不可见，缺少状态、权限和审计。

## 竞品 B：Automation Builder

- 定位：把外部 SaaS 串成自动化流程。
- 功能：触发器、步骤编排、日志和失败重试。
- 局限：LLM 节点通常是黑盒 prompt，缺少中间思考和结构化输出。

## 竞品 C：AI Agent Workspace

- 定位：让员工通过聊天调用工具。
- 功能：文件搜索、网页总结、工具调用、团队知识库。
- 局限：复用模板弱，过程难以产品化展示。
