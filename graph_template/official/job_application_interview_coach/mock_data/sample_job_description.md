# 示例目标 JD

## AI Workflow Product Engineer

团队正在建设面向企业内部知识和流程自动化的 AI 工作流平台，需要候选人同时理解前端产品体验、后端接口、可观测性和 LLM 应用边界。

### 职责

- 设计和实现工作流编辑器、运行记录、权限确认和模板 Gallery。
- 与产品和算法团队合作，把 LLM 能力封装成可审计、可复跑的业务流程。
- 建设端到端测试、运行追踪和错误诊断工具。
- 优化企业用户在知识库检索、模板复用和人工审核中的体验。

### 要求

- 3 年以上 TypeScript 前端或全栈经验。
- 熟悉 Vue 或 React，能独立负责复杂交互和状态管理。
- 有 Node.js/API、数据库或后端协作经验。
- 理解 LLM/RAG/Agent 工作流中的常见风险，例如幻觉、权限、审计和上下文预算。
- 能用数据或真实案例说明产品效果。

### 加分

- 有低代码、工作流、知识库、AI 工具或企业协作产品经验。
- 有 Playwright、可观测性、权限系统或多语言产品经验。
