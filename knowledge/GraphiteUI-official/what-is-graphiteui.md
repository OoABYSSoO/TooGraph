# 什么是 GraphiteUI

GraphiteUI 是一个围绕节点编排、技能调用和结构化运行结果构建的工作流界面原型。

它的目标不是只让用户“调用一次模型”，而是把输入、agent、condition、output 这些节点组织成一条可观察、可修改、可重复执行的流程。用户可以在同一张画布里查看输入来源、节点连接关系、输出结果以及运行状态。

GraphiteUI 当前重点展示三类能力：

- 节点式编排：把输入、处理中间节点和输出连接成一条流程。
- skill 能力：在 agent 节点里挂载已有 skill，把外部能力接进流程。
- grounded answer：通过知识库检索结果为 agent 提供上下文，让回答更接近官方资料而不是自由发挥。
