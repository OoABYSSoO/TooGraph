# TooGraph Context Fanout

Read-only Buddy context assembly. This action fans out across Buddy Home MEMORY.md,
knowledge retrieval, current page context, and discoverable capability candidates
from templates, enabled Actions, and Tools. It returns one budgeted
`context_brief` plus an auditable merge report.

It must not write files, Buddy Home, graph state, settings, templates, or runtime
configuration. All outputs are context only and do not grant permissions.
